# MaxKB 知识库构建指南

## 概述

开源企业级 RAG 平台（13k+ GitHub Stars），开箱即用的知识库问答系统。

## 核心特性

- 🚀 **零编码集成** — 上传文档 → 自动拆分/向量化 → RAG 问答
- 🤖 **多模型支持** — 可接入 OpenAI、通义千问等多种大模型
- 🔧 **MCP 工具调用** — 支持通过 MCP 协议调用外部工具
- 🏢 **企业级特性** — 权限管理、审计日志、高可用部署

## 使用流程

1. 创建知识库
2. 上传文档（支持 PDF/Word/Excel/Markdown 等）
3. 系统自动拆分与向量化
4. 配置检索参数
5. 绑定到应用，开始问答

## 适用场景

- 企业内部知识库
- 客服自动问答
- 技术文档检索
- 培训材料管理
