# RAGFlow 知识库构建指南

## 概述

开源 RAG 引擎（35k+ GitHub Stars），基于深度文档理解（DeepDoc），适合复杂格式文档处理。

## 核心特性

### DeepDoc 深度文档理解
- 📄 **布局识别** — 自动识别标题、正文、图表等版面元素
- 🔍 **OCR 支持** — 扫描件、图片中的文字提取
- 📊 **表格结构识别** — 自动解析表格结构和数据
- 📝 **15+ 种切片方法** — 按段落/标题/页面/语义等多种方式切分

## 与其他方案对比

| 特性 | RAGFlow | Dify | MaxKB |
|------|---------|------|-------|

| 切片方法 | 15+ | 2 种 | 自动 |
| OCR | ✅ 原生支持 | ❌ | ❌ |
| 表格识别 | ✅ | ❌ | ❌ |
| 上手难度 | 中等 | 简单 | 简单 |
| 适用场景 | 复杂格式文档 | 通用问答 | 企业快速部署 |

## 适用场景

- PDF 文档/学术论文
- 扫描件/图片文档
- 包含表格的复杂报告
- 需要精确版面理解的场景
