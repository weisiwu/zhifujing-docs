# Token 中转站深度解析

## 概述

分析大模型生态中的中间层——从合规 AI Gateway 到灰色 Token 转售的边界。

## 两种形态

### 合规 AI Gateway
| 功能 | 说明 |
|------|------|
| 统一接口 | 一个 API 对接多家模型 |
| 智能路由 | 根据任务类型自动选择模型 |
| Fallback | 主模型不可用时自动切换 |
| 预算治理 | 用量控制、费用预警 |
| 密钥管理 | 统一管理 API Key |

### 灰色 Token 转售
- 利用账号注册优惠批量获取 Token
- 以低价转售给第三方用户
- 存在封号、数据泄露等风险

## 技术架构

```
用户请求 → AI Gateway → 路由决策 → 模型调用
                ↓
          预算检查 → Fallback → 日志/审计
```

## 关键洞察

> AI Gateway 的核心价值不是「中间商赚差价」，而是统一管理、智能路由和成本控制。好的 Gateway 应该让企业用得更安全、更省钱。
